Description

Pizza so good it will make you want to slap your momma!

Fresh, cheesy, and with the perfect crust!

Ingredients

  • Great Value Cheese
  • Great Value Premade Pizza Crust
  • Great Value Pepperoni
  • Great Value Pizza Sauce
  • Any other cheap great value brand toppings you prefer!

Steps

  1. Grab your crust and take it out the package
  2. Slap the sauce down on it
  3. Top it with cheese and other toppings
  4. Sling it in the oven at 300F for 18 Minutes
  5. If it doesn't taste good. Buy Papa Johns!
